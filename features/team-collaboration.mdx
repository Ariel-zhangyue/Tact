万有无界团队协作功能为企业提供了一套完整的实时沟通解决方案，涵盖频道消息、@提及、文件共享、表情回应、消息搜索等核心能力。通过统一的工作台，您和团队成员可以在结构化的频道中高效交流，减少信息分散和沟通断层。无论是跨部门项目协同、日常团队讨论，还是一对一即时沟通，万有无界都能帮助您的团队保持信息对齐、快速响应。

## 创建和管理频道

频道是团队沟通的核心单元，用于围绕特定项目、主题或部门展开讨论。

**创建频道**

1. 在左侧导航栏的「频道」区域，点击右上角的 **＋** 按钮，选择 **创建频道**。
2. 输入频道名称（建议使用简洁、易识别的名称，如 `#市场-Q3活动` 或 `#技术-后端开发`）。
3. 选择频道类型：
   - **公开频道**：企业内所有成员均可搜索并加入，适合全员公告或跨团队协作。
   - **私有频道**：仅受邀成员可见和加入，适合涉及保密内容的项目或小组讨论。
4. 填写 **频道描述**，简要说明该频道的用途，帮助新成员快速了解讨论范围。
5. 点击 **创建** 完成操作。

**添加成员**

进入频道后，点击右上角的 **成员管理** 图标，在搜索框中输入姓名或工号，找到目标成员后点击 **添加** 即可。您也可以在创建频道时批量邀请初始成员。

**设置频道主题与描述**

频道管理员可以点击频道顶部的名称或设置图标，进入 **频道详情** 页面，随时更新 **频道主题**（显示在频道顶部，适合提示当前阶段的重点任务）和 **频道描述**（介绍频道的长期用途）。

<Tip>
在频道名称前使用 # 标识，可让团队成员快速识别工作频道。
</Tip>

## 发送消息

**输入和发送消息**

在频道或私信窗口底部的输入框中输入消息内容，按 **Enter** 键发送。如需换行，可使用 **Shift + Enter**。

**@提及团队成员**

在消息中输入 **@** 并紧跟成员姓名或工号，从下拉列表中选择目标成员，即可完成提及。被 @提及的成员将收到专属通知提醒，确保重要信息不被遗漏。您也可以使用 **@所有人** 向频道内全体成员发送通知，请谨慎使用以避免打扰。

**发送文件和图片**

点击输入框左侧的 **附件** 图标（回形针），从本地选择文件或图片上传。您也可以直接将文件拖拽至对话窗口完成发送。图片将在消息中内联预览，其他文件类型将以附件卡片的形式展示。

**表情回应**

将鼠标悬停在任意一条消息上，点击出现的 **😊** 表情图标，从表情选择器中选择您的回应。表情回应会累计显示在消息下方，适合快速确认、点赞或表达态度，而无需单独回复一条消息。

**在消息线程中回复**

将鼠标悬停在消息上，点击 **回复** 图标，即可在该消息的专属线程（Thread）中展开讨论。线程回复不会打扰频道主流，适合针对某条消息深入探讨，同时保持频道整体简洁。

## 文件共享

**上传文件**

您可以通过以下两种方式在频道或私信中共享文件：

- **拖拽上传**：直接将文件从本地文件管理器拖拽至聊天窗口，松手后文件将自动上传并发送。
- **点击上传**：点击输入框旁的附件图标，在本地选择文件后确认上传。

**支持的文件格式**

万有无界支持上传和预览多种主流格式，包括：

| 类别 | 支持格式 |
|------|----------|
| 图片 | JPG、PNG、GIF、WebP、SVG |
| 文档 | PDF、Word（.doc/.docx）、PowerPoint（.ppt/.pptx）|
| 表格 | Excel（.xls/.xlsx）、CSV |
| 压缩包 | ZIP、RAR |
| 其他 | TXT、Markdown、代码文件等 |

**文件预览与下载**

上传后，支持在线预览的文件（如图片、PDF、Office 文档）会在聊天窗口中直接展示预览卡片，点击可全屏查看。点击预览卡片右上角的 **下载** 图标，可将文件保存至本地。

<Info>
文件上传单次最大支持 100 MB，超大文件建议通过「文档管理」模块上传。
</Info>

## 消息搜索

万有无界提供强大的全局消息搜索功能，帮助您快速定位历史沟通记录。

**基础搜索**

点击界面顶部的 **搜索框**（放大镜图标），输入关键词后按 **Enter**，系统将在所有您有权限访问的频道和私信中搜索包含该关键词的消息。

**按发件人筛选**

在搜索结果页面，点击 **筛选** 按钮，选择 **发件人**，输入团队成员姓名，即可只查看该成员发送的消息，快速找到特定人员的历史发言。

**按日期范围筛选**

在筛选面板中选择 **日期范围**，设置起始和截止日期，将搜索结果限定在特定时间段内，适合追溯某个项目阶段内的沟通记录。

**在特定频道内搜索**

进入目标频道后，使用顶部搜索框，默认会优先在当前频道内进行搜索，减少无关结果的干扰。您也可以在搜索结果页手动选择 **限定频道** 进一步过滤。
